# Phase 1 — Page Priority and Wireframe Order

_Order in which pages are wireframed → designed → built. Driven by the Phase 0 priority brief._

1. **Home** — positioning + 4 differentiators + live demo; entry point for the buyer funnel.
2. **Contact** — primary B2B conversion (Book a demo / Request access); highest business value.
3. **Stock Research** — capability demo #1 (broadest appeal; all brokers score 5 on stock info, so this
   must clearly show our *cited + multilingual* edge).
4. **NALCO Intelligence** — capability demo #2 (the filing-explanation wedge; sample data until sources).
5. **Algo Education** — capability demo #3 (seed content already provided: NSE FAQ + empanelled vendors).
6. **Pricing** — DEFERRED (nav slot reserved; build when business model exists).

**Cross-cutting (designed alongside, not a separate page):**
- Floating widget + full-screen + mobile assistant states.
- Global header/footer, language switcher, compliance/disclaimer.

**Rationale:** Convert-path first (Home → Contact), then the three proof pillars in order of breadth
and content-readiness. Each capability page reuses the same demo-panel + citation components.
