# Phase 1 — Visual Reference Board

_No brand exists yet. This sets a starting direction for hi-fi design; owner to confirm/adjust._

## Style direction
- **Tone:** trustworthy, modern fintech — credible enough for compliance, friendly enough for Tier-2/3
  investors. Clean, lots of whitespace, confident typography.
- **Color (proposed, to confirm):** deep trust-blue/navy primary; a single warm accent for CTAs;
  neutral grays; success-green and caution-amber reserved for citations/disclaimers. WCAG AA contrast.
- **Typography:** one type system covering **Latin + Devanagari + Tamil** with consistent weight/feel
  (e.g., a Noto-family pairing) so EN/HI/TA pages look like one product. Generous line-height for Indic
  scripts; test long Tamil strings for truncation.
- **Imagery:** product-UI screenshots over stock photos; show the assistant answering with citations.
- **Components:** rounded cards, soft shadows, clear source/citation chips, distinct disclaimer style.

## Inspirations
- **Intercom / Fin** — clean B2B SaaS marketing + product-demo storytelling.
- **Groww** — approachable, mobile-first Indian retail finance UX.
- **Stripe-style docs/embed** — "add one snippet" developer-friendly embed framing.
- **Linear/Vercel** — crisp typography, whitespace, dark-accent confidence.

## Anti-inspirations
- Cluttered, jargon-heavy legacy broker portals (dense tables, tiny text).
- English-only "translate" toggles that break Indic-script layout.
- AI chat UIs with no source/citation — the exact trust gap we differentiate on.
- Over-animated, gimmicky bots that hide the human-handoff path.
